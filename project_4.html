Infrastructure Maintenance and
NetBackup Client Upgrade

Project Details

Client: Hatton National Bank (HNB)
Location: HNB Head Office / Data Center and DR Site
Period: January 2025 – April 2025

1. Project Overview

This engagement involved a series of critical infrastructure maintenance tasks and upgrades to ensure the high availability and reliability of HNB’s banking systems. The scope ranged from facilitating a major hardware migration (Hitachi to Dell servers) to executing software upgrades for data protection (NetBackup) and database performance (Exadata). It also included routine but essential hardware maintenance, such as storage battery replacements, to safeguard data integrity.

2. Technical Scope

Data Protection: Upgrading Veritas NetBackup client software across production and DR environments.
Server Infrastructure: Facilitating the migration of virtualized workloads from legacy Hitachi servers to new Dell servers.
Engineered Systems: Patch management for Oracle Exadata systems to maintain security and performance.
Storage Maintenance: Hardware servicing of Hitachi Storage systems to prevent data loss during power events.

3. My Contributions

Phase 1: Server Migration Support

  • Data Collection & Planning: Gathered critical system data to facilitate the replacement of an old Hitachi server with a new Dell server.
  • Configuration Analysis: Compiled details on VMs, LUNs, and system configurations for the migration team to ensure a seamless transition.

Phase 2: Oracle Exadata Patching

  • Patch Upgrade: Executed an Exadata patch upgrade at the HNB site.
  • Coordination: Coordinated between Oracle support and HNB to ensure successful execution without disrupting banking operations.

Phase 3: Storage System Maintenance

  • Battery Replacement: Replaced batteries in Hitachi storage systems at the HNB Tower Data Center.
  • Data Integrity Assurance: Verified battery functionality to ensure cache memory backup power during power failures.

Phase 4: NetBackup Client Upgrade

  • Execution: Performed NetBackup client upgrade starting with the DR site, ensuring correct configuration.
  • Validation: Conducted post-upgrade tests on backup and restore functionalities.

4. Technologies & Tools Used

Veritas NetBackup Oracle Exadata Hitachi Storage Dell Servers Virtualization
