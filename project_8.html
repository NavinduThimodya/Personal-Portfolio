1. Project Overview

This comprehensive engagement involved both the deployment of new infrastructure and the critical maintenance of existing systems. The core project focused on the physical deployment of a new server farm at the Piliyandala Data Center, requiring precise rack allocation and structured cabling. Parallel to this, I executed capacity expansions for storage appliances and performed component-level hardware replacements on high-performance Huawei and Hitachi systems to ensure operational continuity.

2. Technical Scope

Data Center Deployment: Physical positioning, rack optimization, and structured cabling for enterprise servers.
Infrastructure Expansion: Scaling storage capacity on Veritas Flex Appliances through hardware integration and logical configuration.
Hardware Maintenance: Component-level troubleshooting and replacement (CPU, Memory, Batteries) for mission-critical storage and compute hardware.

3. My Contributions

Phase 1: Server Deployment & Structured Cabling

  • Server Positioning: Aligned servers within rack units at Piliyandala DC according to allocation plans.
  • Structured Cabling: Routed network and power cables, separating power/data to prevent EMI and applying strain relief.
  • Labeling & Verification: Implemented rigorous labeling per Dialog standards.
  • Power-Up & Configuration: Powered up servers, installed firmware/software, and performed health checks.

Phase 2: Flex Appliance Expansion

  • Expansion Execution: Carried out storage expansion for a Flex Appliance at Piliyandala DC.
  • Configuration: Connected new expansion unit in ring connection and re-configured storage pool via system topology console.

Phase 3: Hardware Component Maintenance

  • Hitachi Storage Maintenance: Replaced battery in Hitachi storage system at I0K DC to ensure cache data preservation.
  • Huawei Server Repairs: Performed CPU replacement and memory upgrades on Huawei servers at Piliyandala DC.
