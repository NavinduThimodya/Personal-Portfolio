1. Project Overview

This project involved the end-to-end deployment of the Oracle Database Appliance (ODA), an engineered system designed to provide a cost-effective, all-in-one database solution for enterprise workloads. The scope included the physical installation of server hardware at the Bank of Ceylon data center, followed by the logical configuration of the operating environment and the creation of databases tailored to the client's specific banking requirements.

2. Technical Scope

The project required expertise in both hardware infrastructure and database administration. The Oracle Database Appliance was selected to streamline database management, offering optimized performance for Online Transaction Processing (OLTP) and analytics. The deployment process moved from physical rack mounting to complex software configurations, ensuring the system was "cloud-ready" or optimized for on-premises use depending on the specific deployment model selected (e.g., bare metal or virtualized).

3. My Contributions

Phase 1: Physical Installation

  • Server Mounting: Visited the BOC main branch to perform physical installation, rack-mounting units securely within data center racks.
  • Cabling & Connectivity: Managed cabling processes, connecting network and power cables to integrate servers into existing infrastructure.

Phase 2: System Initialization

  • Connectivity: Established initial secure connections to the ODA server.
  • System Verification: Performed comprehensive health checks on hardware, storage, network, and OS to ensure stability before deployment.

Phase 3: Database Configuration

  • Database Creation: Executed creation via ODA web console and CLI.
  • Configuration: Defined DB Name, Version, Character Set, and Architecture (CDB/Non-CDB).
  • Validation: Monitored deployment and performed post-deployment service verification.
