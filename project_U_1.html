RFID-Based Asset Tracking System
for Forklifts

Project Details

Module: EN 2160 Engineering Design Realization
Role: Individual Contributor

1. Project Overview

Designed and engineered a ruggedized, cost-effective RFID tracking device specifically for industrial forklift operations to address the lack of affordable logistics solutions in the local market. The device utilizes 13.56 MHz NFC technology to identify assets loaded onto forklifts and transmits tracking data to a central system via WiFi using MQTT protocols.

2. Technical Implementation

Electronics & Hardware Design

  • PCB Design: Developed a custom 2-layer PCB integrating an ESP32 WROOM 32 microcontroller and an NXP PN532 NFC controller.
  • Antenna Engineering: Designed a custom PCB antenna with a specific matching circuit and EMC filter to resonate at 13.56 MHz and match the impedance of the PN532 chip for maximum power transfer.
  • Power Management: Engineered a power management circuit using the LM2596S-3.3 voltage regulator to efficiently convert the forkliftâ€™s 12 V battery output to a stable 3.3 V logic level.

Firmware Development (Embedded C)

  • Framework Migration: Migrated firmware development from Arduino IDE to the ESP-IDF framework to utilize register-level programming for optimized hardware control and resource efficiency.
  • Communication Protocols: Implemented SPI communication between the ESP32 and PN532, and established a real-time data pipeline using MQTT to publish asset detection data to a cloud broker.
  • Critical Debugging: Identified and resolved a timing compatibility issue between the Adafruit PN532 library and the ESP32's 240 MHz clock speed by manually adjusting the communication bitrate to 100 kHz to prevent data glitches.

Mechanical Design

  • Rugged Enclosure: Designed a rugged enclosure using SolidWorks, featuring upper housing and lower chassis components tailored for forklift mounting and 3D printed for prototyping.
