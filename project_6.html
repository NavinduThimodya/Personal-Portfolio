1. Project Overview

This project focused on the maintenance and modernization of the client's Veritas Flex Appliance infrastructure. The engagement began with critical troubleshooting of storage hardware failures to restore system redundancy. Subsequently, a comprehensive upgrade of the appliance was executed, covering both firmware and application instances, followed by post-upgrade stabilization to ensure the automation of backup schedules.

2. Technical Scope

Hardware Maintenance: Diagnosing and resolving disk state errors (Foreign/Failed) within the Flex Appliance storage array.
Infrastructure Upgrade: Executing a multi-layered upgrade process involving the Flex Appliance software, firmware, and hosted application instances (Primary, Media, and WORM servers).
Operational Stability: Troubleshooting policy execution failures to restore automated backup operations.

3. My Contributions

Phase 1: Storage Troubleshooting & Recovery

  • Failure Analysis: Identified a disk failure in the Flex Appliance storage array where a disk had entered a "foreign" state.
  • Restoration: Collaborated with Veritas Support to re-configure the disk, successfully restoring it to "online" active status and ensuring full storage health.

Phase 2: Flex Appliance Upgrade

  • Upgrade Execution: Performed a multi-step upgrade of the Flex Appliance software followed by firmware updates.
  • Instance Management: Managed the upgrade of application instances by stopping instances, uploading new versions, executing pre-checks, and committing upgrades for Primary and Media servers.

Phase 3: Post-Upgrade Optimization

  • Issue Resolution: Diagnosed a scheduling conflict where backup policies failed to run automatically post-upgrade and performed necessary configurations to restore the workflow.
