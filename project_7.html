1. Project Overview

This project involved the end-to-end deployment of an enterprise-grade Commvault backup solution to secure the bank's critical data. The scope encompassed the physical installation of backup infrastructure, system modernization through firmware upgrades, and the establishment of a Disaster Recovery (DR) site. The project concluded with rigorous validation and testing to ensure data recoverability across both production and DR environments.

2. Technical Scope

Infrastructure Deployment: Physical rack mounting and cabling of main servers and storage units for the backup infrastructure.
System Modernization: Execution of firmware and software updates to ensure compatibility with the latest Commvault versions.
Disaster Recovery (DR): Establishing a secondary backup site at the Piliyandala Data Center to ensure business continuity.
Validation & Testing: Comprehensive verification of hardware health, network connectivity, and backup integrity.

3. My Contributions

Phase 1: Physical Installation

  • Rack Mounting: Performed physical installation of main backup server and storage servers at NDB facility.
  • DR Site Setup: Initiated creation of the DR site at Piliyandala Data Center, repeating rack mounting and installation for DR storage units.

Phase 2: System Configuration & Upgrades

  • Firmware Management: Performed necessary firmware upgrades and installed updates to ensure hardware compatibility.
  • Compatibility Checks: Optimized the operating environment for Commvault deployment.

Phase 3: Validation and Testing

  • Infrastructure Verification: Inspected hardware components and verified network/logical connections.
  • Backup Testing: Executed test backup jobs for both Production and DR sites to confirm data recoverability.
  • Customer Handover: Assisted in obtaining final customer verification of the deployment.
