AI-Driven Transformer Thermal
Inspection System

Project Details

Role: Full Stack Developer & AI Engineer

1. Project Overview

Developed a comprehensive web-based platform designed to digitize the maintenance workflow of electrical transformers. The system replaces manual inspection processes with an automated pipeline that manages transformer records, visualizes thermal data, and utilizes Artificial Intelligence to detect temperature anomalies such as hotspots and asymmetries.

2. Key Features & Implementation

Full-Stack Architecture

  • Frontend & Backend: Built a responsive React frontend for interactive image analysis and a Spring Boot backend handling RESTful API requests.
  • Data Management: Integrated PostgreSQL to manage transformer metadata, inspection history, and user logs.

AI-Based Anomaly Detection (Phase 2)

  • Detection Engine: Engineered an automated detection engine that compares "Baseline" (historical) vs. "Maintenance" (current) images side-by-side.
  • Algorithms: Implemented a thresholding algorithm (default 10% deviation) to flag thermal anomalies, providing severity scores and confidence levels.
  • Configurability: Created a configurable "Error Ruleset" allowing engineers to dynamically adjust detection sensitivity.

Human-in-the-Loop Annotation (Phase 3)

  • Interactive Suite: Developed an interactive annotation suite allowing engineers to resize, reposition, or delete AI-generated bounding boxes.
  • Feedback Loop: Implemented a feedback loop where user-modified annotations are logged to validate and retrain the model for improved future accuracy.

Digital Record Generation (Phase 4)

  • Automation: Automated the creation of formal Maintenance Record Forms, combining system-generated anomaly data with manual engineer inputs (status, electrical readings).

3. Technologies & Tools Used
